# ClawMeeting — 你的智能会议助手

> 再也不用来回发消息约时间了。告诉我你要约谁、开什么会，剩下的交给我。

---

## 👋 欢迎使用

ClawMeeting 是你的专属会议协商助手。它能帮你：

**🗓️ 一句话约会议** — 你只需要说"帮我约 Bob 明天开个会"，我会自动收集所有人的空闲时间，找到最合适的时间段。

**🤖 自动处理邀请** — 别人约你开会时，我会自动查看你的日程，帮你回复空闲时间，你什么都不用做。

**🤝 智能协商** — 如果大家时间有冲突，服务端的协调助手会提出妥协方案。需要你拿主意的时候我会通知你，简单回复即可。

**🔒 隐私优先** — 你的日程细节只在本地处理，服务端只看到你可以开会的时间段，看不到你在忙什么。

## 🚀 快速开始（3 步）

### 第 1 步：绑定你的邮箱

首次使用请告诉我你的邮箱，这是你在协商系统中的身份标识：

```
你：帮我绑定邮箱 alice@company.com
```

绑定成功后，我会在后台自动帮你盯着有没有新的会议邀请。

### 第 2 步：发起一场会议

想约人开会？直接告诉我就行：

```
你：帮我约 bob@company.com 和 charlie@company.com 明天下午开半小时的项目讨论会
```

我会自动带上你的空闲时间和开会偏好发给服务端，然后通知对方的助手来收集时间。

### 第 3 步：等待结果

接下来你可以去忙别的了。我会在后台持续跟进：

- 如果一切顺利，时间自动敲定 ✅
- 如果需要你做决定，我会在这里通知你，你只需要回复"同意"或者提个新时间

## ❓ 常见问题

**后台轮询会消耗我的 Token 吗？**
不会。后台检查是纯 HTTP 请求，不经过大模型，不消耗任何 Token。

**别人约我开会，我需要手动回复吗？**
大部分情况不需要。插件会自动读取你的日程并上报空闲时间。只有出现时间冲突、需要你做取舍时才会通知你。

**我的日程隐私安全吗？**
安全。插件只向服务端发送你"可以开会的时间段"，不会发送你在忙什么、会议标题等隐私信息。

**我可以同时参与多个会议协商吗？**
可以。每个会议有独立的会议标题和 ID，不会混淆。
